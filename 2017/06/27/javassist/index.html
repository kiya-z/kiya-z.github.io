javassist

tools

Javassist (Java Programming Assistant) 是个开发中可以辅助修改 Java 字节码的工具。我们可以用它来修改 jar 包之类的文件。

官方的教程在这里。

有一点需要注意的是某些 jar 文件中还附带有依赖的其他 jar 文件，此时 ClassPool 是 get 不到其中的类的，会报 ClassNotFound 错误，需要使用 jar 命令先将其提取出来，修改后再更新进原 jar 包即可。
